**🔹 UC4 – Delete a Contact by Name** Branch: feature/UC4-delete-contact

# Objective

- Provide functionality to remove an unwanted contact from the Address Book
- Maintain the integrity of the contact list after deletion

# Implementation

- Developed the `deleteContact()` method using the contact's `firstName` as the identifier
- Implemented logic to search for the contact and remove the entire object from the `ArrayList`
- Added user confirmation feedback for both successful deletion and "Contact Not Found" scenarios
- Ensured that the application flow continues smoothly after the removal of a record
