**🔹 UC3 – Edit Existing Contact Details** Branch: feature/UC3-edit-contact

# Objective

- Allow users to modify existing contact information using their name
- Ensure the system can locate and update specific records accurately

# Implementation

- Implemented the `editContact()` method to search for a contact by `firstName`
- Integrated a sub-menu to allow users to choose which specific field to update (e.g., Address, City, Phone)
- Used setter methods to update the contact details in the `ArrayList`
- Added logic to display a "Contact Not Found" message if the search query returns no results
